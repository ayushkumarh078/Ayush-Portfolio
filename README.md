# Ayush Kumar — Portfolio LDR-01

A cyberpunk-inspired portfolio website themed on *Love, Death + Robots* visual language. Single-page, zero-build, vanilla HTML/CSS/JS.

## Tech Stack
- HTML5, CSS3 (custom properties, `clamp()`, grid/flexbox)
- Vanilla JavaScript (ES6)
- Google Fonts: Anton, Inter, JetBrains Mono

## Deployment
Deployed via GitHub Pages from `main` branch root. No build step required.
